# 🍕 How Much Pizza?

Stop guessing how much food to order. Plug in your group size, slide the hunger scale, get a recommendation.

## Features

- **5 food types** — Pizza, tacos, subs, wings, Chinese
- **Adults & kids** — Different appetites, different math
- **Hunger scale** — From "we already ate" to "post-workout football team"
- **Side dish adjustment** — Getting sides? You need less of the main
- **Leftovers toggle** — Because cold pizza tomorrow is a feature, not a bug
- **Smart tips** — Context-aware suggestions for your specific order

## Quick Start

```bash
npm install
npm run dev
```

## Tech

React + TypeScript + Vite. No backend needed — all calculation is client-side.
